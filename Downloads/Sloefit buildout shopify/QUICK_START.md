### 3. Essential Settings

#### Announcement Bar
- Click **"Announcement Bar"** section
- Edit the message text
- Add a link if you want it clickable

#### Header
- Click **"Header"** section
- Upload your logo (or it will show "Sloe Fit" text)
- Select your main menu from the dropdown

#### Hero Section
- Click **"Hero"** section
- Edit the heading: "Your Complete Fitness Ecosystem."
- Edit the text
- Set button links:
  - Primary: Link to your collection (e.g., `/collections/starter-stack`)
  - Secondary: Link to AI Coach page (e.g., `/pages/ai-coach`)

#### Featured Collection
- Click **"Featured Collection"** section
- **IMPORTANT:** Select your collection (create "The Starter Stack" collection first!)
- Set how many products to show (default: 3)

#### How It Works
- Click **"How It Works"** section
- You'll see 3 "Step" blocks
- Click each block to edit:
  - Number (01., 02., 03.)
  - Heading
  - Text

#### Brand Story
- Click **"Brand Story"** section
- Upload an image
- Edit the title and text
- Set the button link (e.g., `/pages/our-story`)

#### Newsletter
- Click **"Newsletter"** section
- Edit the title and text
- The form automatically works with Shopify's email system!

#### Footer
- Click **"Footer"** section
- You'll see 4 column blocks
- Edit each column:
  - First column: Logo text and mission statement
  - Other columns: Select menus (create these in Navigation first!)

### 4. Create Your Collections
1. Go to **Products** â†’ **Collections**
2. Click **"Create collection"**
3. Name it **"The Starter Stack"**
4. Add your products to it
5. The Featured Collection section will automatically show these products!

### 5. Set Up Navigation Menus
1. Go to **Online Store** â†’ **Navigation**
2. Create menus:
   - **Main menu:** Shop, The AI Coach, Our Story, The Guide
   - **Footer menus:** Shop links, Company links, Support links
3. Assign the main menu in the Header section
4. Assign footer menus in the Footer section

### 6. Configure Theme Settings
1. In Theme Editor, click **"Theme settings"** (bottom left)
2. Set your colors:
   - Primary Accent: `#C8FF00` (Electric Lime)
   - Background: `#121212` (Dark)
   - Text: `#FFFFFF` (White)
3. Choose fonts (Inter recommended)
4. Adjust page width if needed

### Products Not Showing?
- Make sure you've created the collection
- Check that products are added to the collection
- Verify the collection is selected in the Featured Collection section

### Menu Not Showing?
- Create the menu in Navigation first
- Then select it in the Header or Footer section settings

### Colors Not Right?
- Go to Theme Settings â†’ Colors
- Or edit `config/theme.json` directly

### Can't Find a Section?
- All sections are in the Theme Editor sidebar
- Click **"Add section"** to add more
- Drag sections to reorder them
